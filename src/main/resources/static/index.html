Hello!

Ready for a new You? We thought of this project as a way to help us achieve those New Year's Resolutions we keep drafting but never really accomplish.

Check & Streak is a simple web service that helps you set and track your goals expanding your streaks based on how often you want to perform these tasks to form new habits.
